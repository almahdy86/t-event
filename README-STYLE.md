# هوية وتوجيهات التصميم — التنفيذي (مختصر)

ملف `styles/design-tokens.css` يحوي المتغيرات الأساسية (ألوان RGB، ظلال، أحجام أزرار).

- ألوان أساسية:
  - `--tanfeethi-brown` — لون التركيز الرئيسي للأزرار والنصوص المهمة.
  - `--tanfeethi-turquoise` — لون داعم لحالات التركيز والـ accents.
  - `--tanfeethi-cream` — لون الخلفية العام.

- قواعد سريعة للاستخدام:
  - استخدم `.btn-primary` للأزرار الأساسية، `.btn-secondary` للثانوي، و`.btn-ghost` للروابط البسيطة.
  - استخدم `Card` للمحتوى البصري مع `card-media` للصور.
  - حافظ على تناسب التباين: نص عادي يجب أن يكون عند 4.5:1 على الخلفية.

نماذج استيراد:

- استيراد الأنماط في `_app.js` يتم تلقائياً عبر `styles/globals.css`.
- أمثلة مكونات: `components/Button.js`, `components/Card.js`.

إذا تريد، أطبق الأنماط على صفحات محددة (مثلاً `pages/index.js`) أو أعدل عناصر الواجهة القائمة لتستخدم هذه المكونات.
